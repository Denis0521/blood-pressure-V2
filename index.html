📖 使用說明
🩺 核心功能
  • 記錄每日 ☀️ 早上與 🌙 晚上的血壓數值
  • 點擊列表右側的「✏️」即可補填或修改
  • 支援日期區間搜尋，追蹤健康變化
  • 一鍵產出乾淨的 PDF，方便回診
☁️ 雲端備份教學
  1. 前往 GitHub 註冊免費帳號
  2. 點選頭像 → Settings → Developer settings
  3. Personal access tokens → Tokens (classic)
  4. Generate new token → Note 寫「血壓備份」
  5. Expiration 選 No expiration
  6. Scopes 只需勾選「gist」
  7. 產生後複製 Token，貼回此 App 即可連線！
