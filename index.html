📖 使用說明書

🩺 核心功能

  • 記錄每日 ☀️ 早上與 🌙 晚上的血壓數值
  • 點擊列表右側的「✏️」即可隨時補填或修改
  • 每次點擊儲存，系統將自動進行雲端安全覆蓋
  • 一鍵產出乾淨的 PDF，方便回診給醫師看

☁️ 雲端備份教學

  1. 前往 GitHub 註冊免費帳號並登入
  2. 點選右上角個人頭像 → Settings
  3. 左側選單最下方 → Developer settings
  4. Personal access tokens → Tokens (classic)
  5. Generate new token (classic)
  6. Note 填「血壓備份」，期限選 No expiration
  7. 權限只勾選 gist
  8. 產生後立刻複製 Token，貼回此 App

☁️ GitHub Gist 雲端同步

尚未設定雲端備份
  1. 前往 GitHub → Settings → Developer settings → Tokens (classic)
  2. 點 Generate new token (classic) → 期限選無限制、只勾選「gist」
  3. 產生後立刻複製 Token，貼到下方進行安全綁定。
