📖 使用說明書
🩺 核心功能
  • 記錄每日 ☀️ 早上與 🌙 晚上的血壓數值
  • 點擊列表右側的「✏️」即可隨時補填或修改
  • 每次點擊儲存，系統將完全自動進行雲端安全覆蓋
  • 一鍵產出乾淨的 PDF，方便回診給醫師看
☁️ 雲端備份教學
  1. 前往 GitHub 註冊免費帳號並登入
  2. 點選頭像 → 選擇 Settings
  3. 滑到最下方 → 點選 Developer settings
  4. 展開 Personal access tokens → 點選 Tokens (classic)
  5. 點擊 Generate new token → ⚠️ 選擇 Generate new token (classic)
  6. 期限 (Expiration) 選 No expiration，並勾選 gist
  7. 滑至最底部點擊 Generate token，產生後立刻複製貼回 App！
